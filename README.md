# Student-Management-System
Project Overview

The Student Management System is a Java-based application designed to manage and maintain student records efficiently. It allows users to add, update, delete, search, and display student information using a structured and user-friendly interface. The system is developed as an academic project to demonstrate object-oriented programming concepts and data management in Java.

# Features
Add new student records
Update existing student details
Delete student records
Search students by ID or name
Display all students in a list or table
Input validation to prevent errors
Simple and clear user interface (Console or GUI)
